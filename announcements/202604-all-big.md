## 程序员生存指南

### 如何优雅地摸鱼

资深前辈总结的 *终极技巧*：

- 打开 `htop`，盯着看，没人敢打扰你
- IDE 字体调成 **12px**，别人凑过来也看不清
- ~~把游戏窗口~~把终端窗口放在最上层
- 学会在任何对话中加入"__这个需要评估一下__"

```
$ git commit -m "fix: 修了个 bug"
$ git commit -m "fix: 修了修 bug 引入的 bug"
$ git commit -m "fix: 不如重写"
$ git commit -m "revert: 算了还是别重写了"
```

> 有人问我写代码最重要的是什么，我说是 `Ctrl+Z`。

## 甲方语录大赏

### 经典名言

**甲方**：这个需求很简单，你们做一下就好了。

翻译如下：

- "简单改一下" = 推翻重做
- "参考一下竞品" = ~~抄一个~~ *致敬*
- "我也不知道要什么，你先做一版" = __你做十版我也不知道要什么__
- "这个颜色能不能再 ~~好看~~ *五彩斑斓的黑* 一点"

> 世界上最遥远的距离，不是生与死，而是甲方说的"**就差一个程序员了**"和项目实际需要的人力之间的差距。

### 白粥

很久很久以前有个老猎人，山中独居数十年，无妻无子，子然一身。

数月前山中打猎之时，见一白狐掉入陷阱，挣扎不得脱。猎人恻隐之心顿起，遂将白狐放生。

说也奇怪，自那日起，猎人每日回家，灶上便有热气腾腾的**白粥**。猎人大喜，食之味道甚异，*不似凡间之物*。

### 赎身

大家好，我是**林翩翩**。

今天刷到一个帖子，说扬州城赎一个红倌人要多少银两。下面有人回：*一百两*。

一百两什么概念呢。方知宕去二十四桥看苏怜烟一次，茶资加听曲加过夜，`十余两`。他去了十几次，够赎我两回了。

__他没赎。__

> 后来我每日刷新接客，自己攒。攒到快够的时候，城破了，多铎来了。我想着横竖是死，不如拉个垫背的，就染了天花去找他B点下包。

多铎临死前问我："你就没想过拿这钱跑吗？"

我想了想，好像确实没想过。

可能这就是零笔下好女孩的宿命吧。钱攒够了，人也死了。

不过现在有个好消息。零说成本找回来了，**DLC也有戏**。我想着既然项目都回本了，我林翩翩那一百两赎身银子是不是也该结一下了？

不多要，这么多年我自己辛勤耕耘也攒了 `50`，~~剩下的50你们帮我补齐就行~~。
